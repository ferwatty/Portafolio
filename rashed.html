RASHED & SMITH

El objetivo de este proyecto era diseñar una guía de viajes de lujo para la editorial Rashed & Smith que permitiera transmitir una experiencia a través de imágenes, información y recorridos que fueran atractivos para el viajero. Para esto se eligieron 3 rutas y con base en una investigación previa, crear un concepto editorial para cumplir el objetivo.
